# Definition of Done

- [ ] Задача соответствует PRD/TASKS конкретного сервиса.
- [ ] Перед кодом был согласован подход.
- [ ] Нет расширения scope без подтверждения.
- [ ] Добавлены/обновлены тесты.
- [ ] Пройдены релевантные команды Maven.
- [ ] Публичные JSON-контракты совместимы.
- [ ] Нет секретов в коде, логах, тестовых данных, artifacts.
- [ ] Ошибки классифицируются через единый `error.type`.
- [ ] Workspace очищается согласно policy.
- [ ] Docker/Kubernetes обновлены, если менялся runtime.
- [ ] Сложная логика документирована Javadoc/комментарием.
- [ ] Итоговый ответ содержит проверки и оставшиеся риски.
