# Frontend MVP

Frontend module for CI/CD master-service console.

## Stack

- React + TypeScript + Vite
- Ant Design
- TanStack Query
- React Router

## Run

```bash
cd frontend
npm install
npm run dev
```

Default URL: `http://localhost:5173`

API base is configured through `.env`:

```bash
VITE_API_BASE_URL=http://localhost:8080/api/v1
VITE_DEFAULT_LOGIN=developer
```
